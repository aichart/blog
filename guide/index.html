# Quick Start

  1. 安装

    npm install
    1

  2. 启动本地服务器（自动打开浏览器进行本地预览）

    npm run docs:dev
    1

  3. 发布

    npm run docs:publish
    1

    TIP

    • 也可以在项目terminal中执行./deploy.sh
    • 自动构建
    • 自动提交到gh-pages分支
    • 可以自定义deploy.sh中的配置
