TIP

  • 对于所有的flex item，可以想象成是flex container中被分割成的一块块的虚拟空间
  • justify-content 在主轴方向上设置flex item的空间分布（分割主轴的空间格子）
  • align-content 在交叉轴方向上设置flex item的空间分布（分割交叉轴的空间格子）
  • align-items 设置交叉轴方向的每个flex元素在虚拟的flex item空间内的对齐方式
  • align-self 类似于align-items, 但它是作用于flex item的且优先级大于align-items
  • flex-grow 作为扩展比例，只有在有多余空间时才会生效
  • float, clear和vertical-align在应用于flex item对象时无效

# Grid

Grid把父容器分割成多个二维空间格子，具体的子元素在这些二维空间格子内进行对齐或拉伸

# grid-template-columns 和 grid-template-rows

格式：[任意名字-可选] <px | 百分比 | fr | auto> | repeat(数量, size [名字-可选])

例如：

.grid {
    grid-template-columns: [c-1] 40px [c-2] 50px [c-3] auto [c-4] 25% [c-5];
    // 同一根line可以有多个名字
    grid-template-rows: [r-start] 25% [r-1-end r-2-start] auto [r-2-end r-3-start] 40px [r-end];
}
.grid2 {
    // 共享同一个名字
    grid-template-columns: repeat(5, 20px [col-start]); 
}
.grid2Item {
    // 共享名字使用格式：名字 line数量（第几个条线）
    grid-column-start: col-start 2;
}

# repeat

格式： repeat(重复次数 | auto-fill, 重复的size-多个size用空格分开)

.grid {
    // 3 列 每列宽度 100px
    grid-template-columns: repeat(3, 100px);
    // 6 列 第1,3,5列宽度为 100px, 第2,4,6列宽度为200px
    // grid-template-columns: repeat(3, 100px 200px);
    // 自动填充：单元格大小固定，容器大小不固定，希望容纳更多的单元格
    // grid-template-columns: repeat(auto-fill, 100px);
}
