# Type Support For .vue Imports in TS（使用Vetur插件）

  1. 安装本地依赖：npm install @vuedx/typescript-plugin-vue --save-dev

  2. 修改tsconfig，compilerOptions -> plugins

    ...
"compilerOptions": {
    "plugins": [
        {
            "name": "@vuedx/typescript-plugin-vue"
        }
    ]
}
...

  3. 删除 src/shims-vue.d.ts （已经不需要它来提供类型支持）

  4. 此时 src/main.ts 会报错

  5. ctrl + shift + p

  6. 搜索Select TypeScript version -> Use workspace version

  7. 报错已经消失，齐活！
